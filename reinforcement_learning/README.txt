Code is available at: https://github.com/dianasaur323/reinforcement_learning

* Requirements *
A working version of Python (I am using Python 3)

* Setup *
1) Create a virtual environment `python3 -m virtualenv venv`
2) Activate the virtual environment `source venv/bin/activate`
3) Install necessary requirements `pip3 install -r requirements.txt`

* Running the program *
1) `python3 main.py`
2) Model name? prompt accepts the following: "t" for taxi data, else frozen lake
